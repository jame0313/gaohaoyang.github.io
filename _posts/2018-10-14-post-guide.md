# 헤더 양식
---
```
---
layout: post //레이아웃 칸, 웬만하면 post 쓰면 된다.
title:  제목
date:   yyyy-mm-dd hh:mm:ss
categories: 카테고리로 카테고리에서 분류할 때 쓴다.
tags: 태그로 태그로 분류할 때 쓴다.
excerpt: 요약, 미리보기 볼때 이 글만 보인다.
mathjax: true or false //수식 쓰면 true 적으면 된다.
---
```
